# JS-LISP

Toy lisp-like language in javascript.

Interactive mode:

    js-lisp> ['+', 1, 2, 3]
    6

## GETTING STARTED

Run interactively:

    $ node js-lisp.js

Run tests:

    $ cd js-lisp
    $ npm install
    $ $ ./node_modules/.bin/nodeunit test/js-lisp.js
